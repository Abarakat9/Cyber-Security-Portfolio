# BSidesKC 2025 — Conference Reflection

A written reflection (ITEC 312) on attending **BSidesKC on April 26, 2025** at Kansas City Kansas Community College — a professional security conference with roughly 150–200 attendees.

## What it covers

- **Practitioner interview** — a conversation with a cloud application security consultant from Orca Security about his career path (networking → security analytics → application security), the certifications he considers most in-demand (Security+, Network+, and a cloud cert like AWS or AZ-900), and his advice for breaking into the field.
- **Keynote takeaways** — notes from a keynote on LLM security risks, covering prompt injection, model abuse, and data leakage, with a mitigation tactic for each (context separation and prompt hardening, output restriction and user verification, strong access control and output monitoring).
- **Honest self-reflection** — including the networking/social side I found harder than expected and what I'd do differently at the next event.

## Why it's here

It documents engagement with the professional security community beyond coursework, exposure to current topics (cloud app security, LLM/AI risk), and self-awareness about professional development — the "encrypt before you launch to the cloud" theme from the event tied directly back to classroom discussion.
