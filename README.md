## `[03]` Featured Projects

### 🛡️ K.E.V.I.N. — Smart Home Security Architecture
**Role:** Security Architect · **Stack:** AWS · Python · ALB · WAF

End-to-end secure smart-home web application. Began as a Python application-layer firewall (`firewall.py`), then re-architected onto a cloud-native edge stack using AWS Application Load Balancer + WAF with managed rule groups including SQLi protection.

`AWS WAF` `ALB` `Python` `Defense in Depth`

---

### 👁️ Facial Recognition AI (ITEC-612)
**Role:** Engineer & Debug Lead · **Stack:** Python · OpenCV · NumPy · Google Colab

Multi-model facial recognition pipeline tested across a 10-celebrity dataset. Debugged dataset URL handling, resolved missing `test_paths` keys in `.npz` artifacts, and refined the PIL/OpenCV preprocessing chain — landing at **100% accuracy across all four models**.

`Python` `OpenCV` `ML Ops` `Image Pipelines`

---

### 🔍 Azure SIEM Threat Detection Lab
**Role:** Solo build · **Stack:** Microsoft Sentinel · KQL · Log Analytics

Honeypot-style detection lab streaming Windows + network logs into Sentinel, with custom KQL analytics rules mapped to MITRE ATT&CK techniques `T1110` (brute force) and `T1071` (C2 traffic).

`Sentinel` `KQL` `MITRE ATT&CK`

---

### 💻 Malicious PowerShell Executions Lab
**Role:** Solo build · **Stack:** Microsoft Sentinel · KQL · Defender for Endpoint

Detection engineering lab focused on identifying adversarial PowerShell behavior — encoded commands, AMSI bypass attempts, suspicious cmdlet chains, and download cradles. Built custom KQL analytics rules mapped to MITRE **T1059.001 (PowerShell)** and tuned alert thresholds to reduce false positives.

`Sentinel` `KQL` `Defender for Endpoint` `T1059.001`
